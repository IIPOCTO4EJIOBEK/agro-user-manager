## 🔌 ПОТОКИ ДАННЫХ

### Сигнализация (WebSocket):
Клиент → Nginx Proxy Manager → 10.0.1.130:80 → jitsi-web-1
                                    ↓
                            10.0.1.130:5280 → jitsi-prosody-1
                                    ↓
                            XMPP WebSocket: /xmpp-websocket
                            BOSH: /http-bind

### Медиа (RTP/UDP):
Клиент ↔ 185.160.36.97:10000 ↔ 10.0.1.131:10000 ↔ jitsi-jvb-1

### Colibri WebSocket (управление видеомостом):
Клиент → NPM → 10.0.1.130:80 → jitsi-web-1 → 10.0.1.131:8080 (JVB)
         /colibri-ws/default/{conference}/{endpoint}

### OCTO (межсерверная связь - отключено в конфиге):
10.0.1.131:10000 ↔ другие серверы JVB
JVB_OCTO_BIND_ADDRESS=10.0.1.131
JVB_OCTO_PUBLIC_ADDRESS=10.0.1.131
