## ⚠️ НАЙДЕННЫЕ ПРОБЛЕМЫ

### 1. Предупреждение в конфиге 8.conf (b24)
```
nginx: [warn] the "listen ... http2" directive is deprecated, 
use the "http2" directive instead in /data/nginx/proxy_host/8.conf:8
```

**Решение:** Заменить строку:
```nginx
# Было:
listen 443 ssl http2;

# Стало:
http2 on;
listen 443 ssl;
```

### 2. Дубликаты в конфиге 10.conf (n8n)
```
proxy_set_header Upgrade $http_upgrade;  # дублируется 2 раза
proxy_http_version 1.1;                   # дублируется 2 раза
```

**Решение:** Удалить дубликаты перед `location / { ... }`
