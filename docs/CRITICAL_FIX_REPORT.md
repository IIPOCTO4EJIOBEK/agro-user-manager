## 🔴 НАЙДЕНА КРИТИЧЕСКАЯ ПРОБЛЕМА

### Конфликт Node.js процессов

**Проблема:**
На веб-узле **10.0.1.220** были запущены процессы **Node.js Push Server** на портах **8010-8015**!

**Почему это неправильно:**
- Push Server должен работать ТОЛЬКО на сервере **10.0.1.230**
- Веб-узлы (220, 221, 222) должны обрабатывать только PHP запросы
- HAProxy направляет WebSocket запросы на **10.0.1.230:8010-8015**
- Когда на 220 тоже есть Node.js на тех же портах - возникает конфликт

**Последствия:**
- Чаты не работают
- WebSocket подключения не проходят
- Перелогин в админке
- Тормоза портала

## ✅ ИСПРАВЛЕНИЕ

