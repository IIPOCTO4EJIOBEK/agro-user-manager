## 🔧 ОПТИМИЗАЦИЯ PROXMOX

### 1. Использовать LXC вместо VM
**Экономия:** 30-40% RAM

```bash
# Создать LXC контейнер вместо VM
pct create 223 local:vztmpl/ubuntu-22.04-standard.tar.gz \
  --rootfs local-zfs:8 \
  --memory 32768 \
  --swap 0 \
  --cores 8 \
  --net0 name=eth0,bridge=vmbr19,ip=10.0.1.223/24 \
  --hostname b24-web-04
```

### 2. ZFS сжатие
**Экономия:** 20-30% диска

```bash
zfs set compression=lz4 nvme_db
zfs set compression=lz4 hdd_mirror
```

### 3. NUMA оптимизация
**Прирост:** 10-15% CPU

```bash
# Для VM с высокой нагрузкой
qm set 200 -numa 1
qm set 200 -cpu cputype=host
```
