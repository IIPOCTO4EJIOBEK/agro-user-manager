### 2. 🟡 REDIS выключен на 10.0.1.221 (Web 02)

**Статус:** `inactive`

**Влияние:** Сессии не синхронизируются между веб-серверами → мобильное приложение получает 401

### Текущая проблема: 401 Unauthorized

**Логи:**
```
83.221.219.226 → 10.0.1.110 → 10.0.1.50 → 10.0.1.220

POST /bitrix/services/main/ajax.php?action=mobile.tourist.remember → 401
POST /mobile/?mobile_action=checkout → 401
```

**Возможные причины:**
1. ❌ Redis session не работает на .221 (выключен)
2. ⚠️ Сессии не синхронизируются между .220/.221/.222
3. ⚠️ Проблема с cookie/аутентификацией

**Диагностика:**
```bash
# Проверить Redis сессии
ssh vardo001@10.0.1.210
redis-cli KEYS "BITRIX_SESSION_*" | head -10

# Проверить сессии на веб-серверах
ssh vardo001@10.0.1.220
ls -la /home/bitrix/www/bitrix/sessions/
```
