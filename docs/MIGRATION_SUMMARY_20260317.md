## 2. Иерархия подчинения (Блок ЭиФ)
- **Цель:** Идеальное дерево для блока Градинаровой М.А.
- **Действие:** Прошит атрибут `Manager` для всей вертикали:
  - Дышлюк Б.А. (ГД) -> Градинарова М.А. (Зам ГД) -> Директора -> Начальники -> Сотрудники.
- **Статус:** ✅ Готово.

## 3. Тотальная очистка систем
### AD 250 (Контроллер домена):
- **Удалено:** **599** объектов (тех. учетки 1С, HealthMailbox, Monitoring, Агроконсалтинг, Президентство).
- **Текущее состояние:** **1616** активных пользователей.

### Bitrix24 (Портал):
- **Действие:** Мгновенная деактивация дубликатов и неактуальных записей через SQL.
- **Логика:** Оставлена только одна (самая свежая) учетная запись для каждого логина из "белого списка" AD 250.
- **Текущее состояние:** **1614** активных пользователей (включая админов и ботов).

## 4. Резервное копирование и безопасность
- **AD Backup:** На сервере 10.0.1.250 создана папка `C:\AD_Snapshot_Experiment`.
- **Состав:** `ntds.dit`, кусты реестра `SYSTEM/SECURITY`, папка `SYSVOL` (GPO).
- **Назначение:** Полное восстановление AD в режиме DSRM в случае сбоя.
